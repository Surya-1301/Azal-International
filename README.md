# Azal International

Premium carpet and floor artistry for luxury hospitality and residential spaces.

## Overview

Azal International is a modern web application showcasing bespoke carpet collections, craftsmanship stories, and custom project capabilities. Built with React and inspired by contemporary luxury brand aesthetics.

## Tech Stack

- **React 18** - UI library
- **Vite** - Build tool and dev server
- **React Router** - Client-side routing
- **CSS** - Custom styling with design tokens

## Getting Started

### Prerequisites

- Node.js 16+ and npm

### Installation

```bash
# Install dependencies
npm install
```

### Run the App

From the project root:

```bash
npm run dev
```

The app will be available at `http://localhost:5173`

### Build

```bash
# Build for production
npm run build

# Preview production build
npm run preview
```
